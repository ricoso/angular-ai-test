# Code Language

## Projekt-Vorgabe
- Code-Sprache: Englisch (FIXIERT)
- REQ-Sprache irrelevant → Code IMMER Englisch

## Rules

1. Code-Sprache = **Setup-fixiert** (siehe Vorgabe oben)
2. REQ-Sprache irrelevant → Code IMMER in Setup-Sprache
3. UI → ALLE konfigurierten Sprachen (siehe `i18n-typings.md`)
4. Naming → Standard English: `loadUsers()`, `onSubmit()`, `User`, `isLoading`

## Deutsch-Beispiele (falls Setup=Deutsch)

| Englisch | Deutsch |
|----------|---------|
| `onSubmit()` | `beimAbsenden()` |
| `loadUsers()` | `ladeBenutzer()` |
| `addUser()` | `fuegeBenutzerHinzu()` |
| `filteredUsers` | `gefilterteBenutzer` |
| `isLoading` | `istLaden` |
| `User` / `CreateUserDTO` | `Benutzer` / `BenutzerErstellenDTO` |
