Echte Marken-SVG-Logos von gottfried-schultz.de werden durchgängig im gesamten UI verwendet, überall wo Marken angezeigt werden: Standortauswahl-Cards, Markenauswahl, Warenkorb/Header, Buchungsübersicht, etc.

**Why:** Konsistentes Branding über alle Wizard-Schritte. Kunde erkennt seine Marke visuell sofort wieder.

**How to apply:** Bei jeder UI-Komponente die Marken anzeigt, die SVG-Logos verwenden statt nur Text. Logos liegen in `assets/brands/` (audi.svg, volkswagen.svg, seat.svg, cupra.svg, skoda.svg, hyundai.svg, porsche.svg, bentley.svg, bugatti.svg, rimac.svg, vw_nutzfahrzeuge.svg).
